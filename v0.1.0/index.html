    Dolang.jl

    Dolang is an implementation of the dolo language and provides utilities for symbolic manipulation and efficient compilation of symbols and expressions into performant Julia functions.

    To read more about the available routines, read the Symbolic manipulation and Dolang's compiler chapters. To get started right away with using the Dolang compiler skip directly to the Examples.
